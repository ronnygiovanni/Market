# Market Order Analysis

A C# console application designed to manage customer orders, calculate discounted totals, and perform basic data analysis to identify top-spending customers and popular products. 

## Features

* **Order Total Calculation:** Computes total prices for customer orders, including an automatic 10% discount for any order total exceeding 500.
* **Customer Analytics:** Utilizes LINQ to evaluate all transactions and identify the "Best Customer" based on total expenditure.
* **Product Tracking:** Aggregates item quantities across all orders to determine the most frequently purchased products.
