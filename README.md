Structura Proiectului (Problema 2)
Codul este impartit in 4 fisiere pentru o mai buna organizare:

OrderItem.cs - Contine datele despre produs: nume, cantitate si pret.

Order.cs - Calculeaza pretul total si aplica reducere de 10% pentru comenzi mai mari de 500.

Items.cs - Include metodele pentru:

GetBestCustomer: Gaseste clientul care a cheltuit cel mai mult.

GetPopularProducts: Calculeaza cate unitati s-au vandut din fiecare produs.

Program.cs - Ruleaza aplicatia si afiseaza rezultatele in consola.
